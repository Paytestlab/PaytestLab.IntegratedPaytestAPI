# Release Notes (Version 1.0.1)

## Overview
This is the first official release, laying the foundation for the project with core functionalities and basic API endpoints.

## Key Changes

### Initial Setup
- Established the project repository and configuration.
- Implemented basic API endpoints:
  - **POST /transaction/start** - Request to start a transaction
  - **POST /transaction/abort** - Request to abort a transaction
  - **GET /status** – Returns the terminal and ECR status.

### Data Structures
- Introduced basic objects.
