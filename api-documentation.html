transaction

Used for controlling transactions

Starts a transaction

header Parameters
X-TestRun-Id
string

Optional identifier for the test run

X-TestCase-Id
string

Optional identifier for the test case

Request Body schema: application/json
required

Starts a new transaction and provides a callback URL for transaction completion notification

transactionId
required
string

Unique identifier for the transaction provided by the client

terminalId
required
string

Terminal ID where the transaction should be started

required
object (TransactionInformation)
callbackUrl
required
string <uri>

URL to notify when the transaction is complete

callbackTimeoutSeconds
integer >= 1

How long (in seconds) the caller will wait for the callback before timing out

Responses

Callbacks

https://127.0.0.1:[port]/transaction/start
